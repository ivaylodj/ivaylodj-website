PROJECT:
The project "ivaylodj-website" is a static personal hobby photographer exhibition website, containing photo galleries, contact/personla info and blog section (decapcms). The photos should not exceed 2048 pxl on the long edge. No photo should be present in more than 1 place i.e. one copy only permitted to save storage! The site source and resources are pushed to GH and then automatically deployed on CloudFlare (CF) Pages Free plan. The domain name is ivaylodj.com. It is currently pointing at a "Coming Soon" website until 26.07.2026 (postponed +48h from 24.07), when the Production should be live.

The project folders and files are:
Local: ~/gh-ivaylodj.com/ivaylodj-website  
GitHub: https://github.com/ivaylodj/ivaylodj-website.git

It is accessed now via ivaylodj-website.pages.dev.

The static website is built onto Aurel photography site template, renamed by me to Cherga. Original template files available locally at ~/html-themes/other\ themes/html/aurel
